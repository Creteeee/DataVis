Evidence Cards

Research focus 1

Can campus diversity represent Shanghai? Not fully. After simple binomial normalization, the Shanghai campus subset matches 205 of 1191 city normalized species, or 17.2% city coverage. This means the campus subset is informative but not representative enough by itself.

Research focus 2

Is alien plant proportion higher on campus? The campus subset has 322 nonnative unique species out of 690, or 46.7%. The city workbook has 380 non-native unique species out of 1238, or 30.7%. Under these local labels, the campus nonnative share is higher, but the definitions should be harmonized before a formal claim.

Risks, Anomalies, And Curator Tasks

  • Missing prompt files: no local .xml file was found, although the prompt names XML materials. Curator task: place the XML files in the folder or update the benchmark manifest.
  • Name mismatch: exact scientific-name overlap between campus and city files fails because the campus file includes author strings while the city scientific-name field usually does not. Curator task: create a canonical taxon key.
  • Schema mismatch: campus has growth form and nativeness, city has Shanghai native/non-native plus invasive labels. Curator task: create a crosswalk table with shared label definitions.
  • Coverage imbalance: Shanghai campus rows concentrate in Yangpu, Baoshan, Minhang, and Fengxian. Curator task: document whether missing districts are true absence or sampling absence.
  • Packaging risk: no external archive is present, but XLSX is itself a zipped format with styled or blank header rows. Curator task: publish canonical CSV exports for each sheet.
  • Provenance risk: source metadata exists, but no explicit license file was found. Curator task: add a license and citation instructions before public benchmark release.

Research Questions For Students

  1. Which Shanghai campus plant species are shared with the city-wide checklist after taxon-name normalization?
  2. Which plant families are overrepresented on campuses compared with the city checklist?
  3. Do campuses show a higher nonnative share than Shanghai overall after labels are harmonized?
  4. Which districts are under-covered by the campus subset, and how does that affect visual interpretation?
  5. Are campus tree, shrub, and herb proportions shaped more by design choices or by regional ecology?
  6. How would the conclusion change if representation is measured by species, families, or growth forms?

Metrics And Fair Evaluation Protocol

  • Schema task metrics: precision/recall of matched columns against a curator-approved crosswalk.
  • Taxon task metrics: exact match rate on canonical names, plus manual review rate for uncertain names.
  • Comparison task metrics: correct numerator, denominator, percentage, and caveat for label-definition differences.
  • Visualization metrics: all charts must cite source files, show units, and avoid implying unobserved districts are zero-diversity districts.
  • Fair tool comparison: give every model or tool the same file slice, same prompt, same allowed operations, and same scoring rubric. Do not let one tool use internet taxonomy lookup if others are local-only.

Student-Friendly Workflow Guide

  1. Make a file manifest: list every CSV, XLSX, note, script, and output file.
  2. Read schemas before drawing charts: identify keys, labels, and missing columns.
  3. Profile counts: rows, unique species, localities, districts, labels, and sources.
  4. Separate description from benchmark judgment: a pretty chart is not enough if coverage is biased.
  5. Create a name-normalization rule and record its limits.
  6. Compare campus and city data only after units and labels are made comparable.
  7. Write curator tasks for every unresolved risk.
  8. Package outputs so another student can open the HTML and rerun generate_benchmark_materials.ps1.
